# Contributing

Story Forge is a solo-author portfolio project. It's public so it can be **read and learned
from**, not because it's soliciting contributions — I'm not accepting feature pull requests.

If you're here to understand how it's built, start with:

- [`README.md`](README.md) — what it is, the demo, and the architecture
- the `AGENTS.md` files (root plus one per major directory) — the conventions that govern the code
- [`story-forge-poc-spec.md`](story-forge-poc-spec.md) — the source of truth for what we build

Found a security issue? Please use the channel in [`SECURITY.md`](SECURITY.md).
