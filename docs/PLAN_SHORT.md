### Decided

- **2026-06-02 — M2.S2 LLM-router decisions, settled via the meta-architect dogfood (ADR 0003; spec §6.5/§6.6 amended).** The architect's `review-architecture` + `decompose-requirement` passes (artefacts: `architecture/reports/2026-06-02-architecture-review.md`, `architecture/proposals/m2s2-llm-router-budget-cap.md`) framed the options; the owner resolved them. The cluster: **(1) Provider order = Ollama → OpenRouter → Grok → Anthropic → Google → OpenAI** — *not* Anthropic-first (overturns ADR 0001). OpenRouter leads the paid routes because one OpenAI-compatible endpoint reaches most vendor models. *Considered & rejected:* Anthropic/OpenAI-primary (ADR 0001's original). **(2) Hand-rolled `httpx` adapters, not vendor SDKs** (D2) — uniform shape, zero new deps, injectable-transport tests, visible swappability; cost = hand-written per-provider usage parsing. *Rejected:* 5 vendor SDKs (5 dependencies to vet/pin/soak, divergent shapes, buries the swappability). **(3) Build only `OllamaProvider` (extended) + `OpenRouterProvider` in M2.S2;** direct vendor adapters deferred to "as needed" (pulls OpenRouter forward from M2.S6). **(4) Control-first budget (G1):** on cloud_free-quota-exhaustion **or** daily-cap, the router **pauses and asks the user** — never silent paid escalation. ## How the two skills use this file

- **`/resume-session`** (session start) reads the **Session handoff** block, verifies the on-disk anchors it names, runs `git log`/`git status`, opens the spec sections listed, and confirms reality matches before work begins.
- **`/wrap-session`** (session end) runs the green-state checks (report-only), checks off completed tasks, strikes (does not delete) obsolete ones, appends a dated line to **Done in previous sessions**, refreshes **Blocked / questions** and **Decided**, and rewrites the **Session handoff** block to point at the next session. It then reminds you to commit on a feature branch (squash-merge hygiene).
- **When a spec change happened:** both skills stop and require `story-forge-poc-spec.md` + `PLAN_LONG.md` to be reconciled with this file before continuing.
